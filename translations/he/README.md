## 🌱 איך להתחיל

הקורס הזה כולל שיעורים המכסים את היסודות של בנייה ופריסה של סוכני בינה מלאכותית.

כל שיעור בונה על הקודם, לכן אנו ממליצים להתחיל מההתחלה ולעבוד עד הסוף.

אם אתם רוצים לחקור עוד נושאים על סוכני AI, תוכלו לבדוק את [קורס סוכני AI למתחילים](https://aka.ms/ai-agents-beginners).

### פגשו לומדים אחרים וקבלו תשובות לשאלותיכם

אם תיתקלו בבעיות או יהיו לכם שאלות לגבי בניית סוכני AI, הצטרפו לערוץ הייעודי שלנו ב-[Discord של Microsoft Foundry](https://discord.gg/Kuaw3ktsu6).

### מה צריך

לכל שיעור יש דוגמת קוד משל עצמה שאפשר להריץ מקומית. תוכלו [לעשות fork למאגר זה](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) כדי ליצור עותק משלכם.

הקורס הזה כרגע משתמש ב:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

אנא ודאו שיש לכם גישה לשירותים האלה לפני תחילת העבודה.

אפשרויות נוספות בנושא אירוח מודלים ושירותים מגיעות בקרוב.
