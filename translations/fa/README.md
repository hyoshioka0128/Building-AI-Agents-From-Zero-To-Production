> **ترجیح می‌دهید به صورت محلی کلون کنید؟**

> این مخزن شامل بیش از ۵۰ ترجمه زبان است که به طور قابل توجهی اندازه دانلود را افزایش می‌دهد. برای کلون بدون ترجمه‌ها، از sparse checkout استفاده کنید:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production.git
> cd Building-AI-Agents-From-Zero-To-Production
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> این به شما همه چیز مورد نیاز برای اتمام دوره با دانلود بسیار سریع‌تر را می‌دهد.

## 🌱 شروع به کار

این دوره شامل درس‌هایی است که اصول ساخت و استقرار عوامل هوش مصنوعی را پوشش می‌دهد.

هر درس بر درس قبلی بنا شده است، بنابراین توصیه می‌کنیم از ابتدا شروع کرده و تا پایان ادامه دهید.

اگر می‌خواهید بیشتر درباره موضوعات عوامل هوش مصنوعی کاوش کنید، می‌توانید به دوره [AI Agents For Beginners](https://aka.ms/ai-agents-beginners) نگاهی بیندازید.

### ملاقات با دیگر یادگیرندگان، پاسخ به سوالات شما

اگر درگیر شدید یا سوالی درباره ساخت عوامل هوش مصنوعی دارید، به کانال اختصاصی ما در دیسکورد [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) بپیوندید.

### آنچه نیاز دارید

هر درس نمونه کد خاص خود را دارد که می‌توانید به صورت محلی اجرا کنید. می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) تا نسخه خود را ایجاد کنید.

در حال حاضر این دوره از موارد زیر استفاده می‌کند:

- [چارچوب عامل مایکروسافت (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [سرویس Azure OpenAI](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [رابط خط فرمان Azure (Azure CLI)](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

لطفاً قبل از شروع اطمینان حاصل کنید که به این خدمات دسترسی دارید.

گزینه‌های بیشتری در زمینه میزبانی مدل‌ها و سرویس‌ها به زودی ارائه خواهد شد.

## مشارکت

این پروژه از مشارکت‌ها و پیشنهادات استقبال می‌کند. بیشتر مشارکت‌ها از شما می‌خواهند تا با  
قرارداد مجوز مشارکت‌کننده (CLA) موافقت کنید که اعلام می‌دارد شما حق دارید و عملاً این اجازه را می‌دهید  
تا حقوق استفاده از مشارکت شما را به ما بدهید. برای جزئیات به <https://cla.opensource.microsoft.com> مراجعه کنید.

هنگام ارسال درخواست کشش (pull request)، یک ربات CLA به طور خودکار تشخیص می‌دهد که آیا نیاز به ارائه  
CLA دارید و درخواست کشش را به‌طور مناسب علامت‌گذاری می‌کند (مثلاً بررسی وضعیت، نظر).  
فقط دستورالعمل‌های ارائه‌شده توسط ربات را دنبال کنید. شما فقط یک بار در تمام مخازن استفاده‌کننده  
از CLA ما باید این کار را انجام دهید.

این پروژه از [کد رفتار متن‌باز مایکروسافت](https://opensource.microsoft.com/codeofconduct/) تبعیت می‌کند.  
برای اطلاعات بیشتر به [پرسش‌های متداول کد رفتار](https://opensource.microsoft.com/codeofconduct/faq/) مراجعه کنید یا  
با [opencode@microsoft.com](mailto:opencode@microsoft.com) برای پرسش‌ها یا نظرات اضافی تماس بگیرید.
