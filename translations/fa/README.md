> **ترجیح می‌دهید محلی کپی کنید؟**

> این مخزن شامل بیش از ۵۰ ترجمه زبان است که حجم دانلود را به طور قابل توجهی افزایش می‌دهد. برای کلون کردن بدون ترجمه‌ها، از sparse checkout استفاده کنید:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production.git
> cd Building-AI-Agents-From-Zero-To-Production
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> این به شما همه چیز لازم برای تکمیل دوره را با دانلود بسیار سریع‌تر می‌دهد.

## 🌱 شروع به کار

این دوره شامل درس‌هایی است که اصول ساخت و استقرار عوامل هوش مصنوعی را پوشش می‌دهد.

هر درس بر درس قبلی بنا شده است، پس توصیه می‌کنیم از ابتدا شروع کنید و تا پایان پیش بروید.

اگر می‌خواهید درباره موضوعات عوامل هوش مصنوعی بیشتر تحقیق کنید، می‌توانید [دوره AI Agents For Beginners](https://aka.ms/ai-agents-beginners) را بررسی کنید.

### آشنایی با دیگر یادگیرندگان، دریافت پاسخ به سوالاتتان

اگر گیر کردید یا سوالی درباره ساخت عوامل هوش مصنوعی داشتید، به کانال اختصاصی ما در دیسکورد بپیوندید در [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### آنچه نیاز دارید

هر درس نمونه کد مخصوص خود را دارد که می‌توانید به صورت محلی اجرا کنید. شما می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) تا نسخه خود را ایجاد کنید.

این دوره در حال حاضر از موارد زیر استفاده می‌کند:

- [چارچوب عامل مایکروسافت (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [خدمات Azure OpenAI](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

لطفاً قبل از شروع، اطمینان حاصل کنید که به این خدمات دسترسی دارید.

گزینه‌های بیشتری درباره میزبانی مدل و خدمات به زودی ارائه خواهد شد.

## مشارکت

این پروژه از مشارکت‌ها و پیشنهادها استقبال می‌کند. برای اکثر مشارکت‌ها لازم است که شما با توافقنامه مجوز مشارکت‌کننده (CLA) موافقت کنید که اعلام می‌کند شما حق دارید و عملاً به ما حقوق استفاده از مشارکت خود را می‌دهید. برای جزئیات، به <https://cla.opensource.microsoft.com> مراجعه کنید.

وقتی که یک درخواست ادغام (Pull Request) ارسال می‌کنید، یک ربات CLA به طور خودکار تعیین می‌کند که آیا نیاز دارید که یک توافقنامه CLA ارائه دهید و درخواست ادغام را به طور مناسب علامت‌گذاری می‌کند (مثلاً بررسی وضعیت، نظر). فقط کافی است دستورالعمل‌های ارائه شده توسط ربات را دنبال کنید. این کار تنها یک بار برای همه مخازن استفاده‌کننده از CLA ما لازم است.

این پروژه از [کد رفتار متن‌باز مایکروسافت](https://opensource.microsoft.com/codeofconduct/) تبعیت می‌کند. برای اطلاعات بیشتر به [سؤالات متداول کد رفتار](https://opensource.microsoft.com/codeofconduct/faq/) مراجعه کنید یا با [opencode@microsoft.com](mailto:opencode@microsoft.com) در تماس باشید برای هر سؤال یا نظر اضافی.
