> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production.git

> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'

> این به شما همه چیز لازم برای تکمیل دوره را با سرعت دانلود بسیار بالاتر می‌دهد.

## 🌱 شروع به کار

این دوره شامل درس‌هایی است که اصول ساخت و استقرار عوامل هوش مصنوعی را پوشش می‌دهد.

هر درس بر درس قبلی ساخته شده است، بنابراین توصیه می‌کنیم از ابتدا شروع کرده و تا پایان ادامه دهید.

اگر می‌خواهید درباره موضوعات عوامل هوش مصنوعی بیشتر کاوش کنید، می‌توانید دوره [AI Agents For Beginners](https://aka.ms/ai-agents-beginners) را مشاهده کنید.

### با یادگیرندگان دیگر ملاقات کنید، سوالات خود را پاسخ بگیرید

اگر گیر کردید یا سوالی درباره ساخت عوامل هوش مصنوعی دارید، به کانال اختصاصی ما در [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) بپیوندید.

### آنچه نیاز دارید

هر درس نمونه کدی خود را دارد که می‌توانید به صورت محلی اجرا کنید. می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) تا نسخه خود را بسازید.

این دوره فعلاً از موارد زیر استفاده می‌کند:

- [چارچوب عامل مایکروسافت (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [سرویس Azure OpenAI](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

لطفاً قبل از شروع مطمئن شوید به این سرویس‌ها دسترسی دارید.

گزینه‌های بیشتر درباره میزبانی مدل و سرویس‌ها به زودی ارائه می‌شود.

## مشارکت

یک توافقنامه مجوز مشارکت‌کننده (CLA) موافقت کنید که اعلام می‌کند شما حق دارید و واقعاً حقوق استفاده از مشارکت خود را به ما اعطا می‌کنید. برای جزئیات، مراجعه کنید به <https://cla.opensource.microsoft.com>.

وقتی یک درخواست کشش ارسال می‌کنید، یک ربات CLA به‌طور خودکار تعیین می‌کند که آیا نیاز دارید یک CLA ارائه دهید
و PR را به‌طور مناسبی علامت‌گذاری می‌کند (برای مثال، بررسی وضعیت، نظر). به سادگی دستورالعمل‌های ارائه شده توسط ربات را دنبال کنید.
شما فقط یک بار باید این کار را در تمام مخازنی که از CLA ما استفاده می‌کنند انجام دهید.

این پروژه از [کد رفتار متن‌باز مایکروسافت](https://opensource.microsoft.com/codeofconduct/) پیروی می‌کند.
برای اطلاعات بیشتر به [سؤالات متداول کد رفتار](https://opensource.microsoft.com/codeofconduct/faq/) مراجعه کنید یا برای هر سؤال یا نظر اضافی با [opencode@microsoft.com](mailto:opencode@microsoft.com) تماس بگیرید.
