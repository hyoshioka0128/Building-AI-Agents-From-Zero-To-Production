## 🌱 開始使用

本課程包含涵蓋建構和部署 AI 代理基礎的課程。

每個課程都建立在前一課的基礎上，所以建議從頭開始，逐步完成到最後。

如果你想深入探索 AI 代理相關主題，可以參考 [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners)。

### 認識其他學習者，獲得問題答案

如果你卡住或有任何關於構建 AI 代理的問題，歡迎加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 的專門 Discord 頻道。

### 你需要準備什麼

每堂課都有自己的程式碼範例，可以在本機執行。你可以[將此倉庫派生](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)創建自己的副本。

本課程目前使用以下項目：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI 服務](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

請確保在開始之前已能存取這些服務。

關於模型託管和服務的更多選項即將推出。
