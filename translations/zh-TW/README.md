## 🌱 開始使用

本課程涵蓋建構與部署 AI 代理人的基礎課程。

每堂課都是由前一堂建立基礎，因此建議從頭開始依序學習至結束。

如果你想進一步探索 AI 代理人主題，可以參考 [AI 代理人入門課程](https://aka.ms/ai-agents-beginners)。

### 認識其他學員，獲得你的問題解答

如果你遇到困難或對建置 AI 代理人有任何問題，歡迎加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 的專屬 Discord 頻道。

### 你所需的條件

每堂課附有可在本機執行的獨立範例程式碼。你可以[在此 fork 這個儲存庫](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)，建立你自己的複本。

本課程目前使用以下項目：

- [Microsoft 代理人框架 (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI 服務](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

請在開始前，確保你有權限使用這些服務。

更多模型託管與服務選項將陸續推出。

## 🗃️ 課程內容

| **課程**                 | **說明**                                                                                                |
|--------------------------|--------------------------------------------------------------------------------------------------------|
| [代理人設計](./lesson-1-agent-design/README.md)           | 介紹我們的「開發者入門」代理人使用案例，及如何設計有效的代理人                                          |
| [代理人開發](./lesson-2-agent-development/README.md)      | 使用 Microsoft 代理人框架 (MAF) 建立三個協助新開發者入門的代理人                                        |
| [代理人評估](./lesson-3-agent-evals/README.md)            | 使用 Microsoft Foundry，評估我們的 AI 代理人表現與改善方式                                           |
| [代理人部署](./lesson-4-agent-deployment/README.md)       | 使用托管代理人與 OpenAI Chatkit，學習如何將 AI 代理人部署至生產環境                                   |
