# धडा 2 एजेंट विकास

"शून्यातून उत्पादनासाठी AI एजंट तयार करणे" या कोर्सच्या दुसऱ्या धड्यामध्ये आपले स्वागत आहे!

या धड्यात आपण पाहणार आहोत:

- आमचे AI एजंट तयार करण्यासाठी लागणारी साधने
  
- आमच्या विकास संसाधनांच्या सेटींग निर्देशांबाबत

- AI एजंट विकासाच्या सर्वोत्तम प्रथा
  
- आमचे AI एजंट तयार करण्यासाठी कोड वॉकथ्रू
  
आता आपण पाहू की AI एजंट तयार करण्यासाठी आपण कोणती साधने वापरणार आहोत.

## साधने आणि सेटींग निर्देश

### मायक्रोसॉफ्ट फाउंड्री

मोठ्या भाषा मॉडेल्स (LLMs) च्या प्रवेशासाठी आपण [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) वापरणार आहोत. फाउंड्री वापरण्यास काही खर्च असतो त्यामुळे जर आपल्याकडे आधीपासून प्रवेश नसेल तर खातं सेटअप कसं करायचं त्याच्या सूचनांचे काटेकोरपणे पालन करा.

### OpenAI मॉडेल्स

या कोर्समधील एजंट कोड नमुने [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) मार्फत OpenAI मॉडेल्स वापरण्यासाठी सेटअप केलेले आहेत.

फाउंड्री वापरून मॉडेल कसे डिप्लॉय करायचे हे जाणून घेण्यासाठी हा मार्गदर्शक वापरा: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

या कोर्ससाठी GPT-4.1 किंवा त्यापेक्षा नंतरचा एक मॉडेल निवडा.

### मायक्रोसॉफ्ट एजंट फ्रेमवर्क

पूर्वी नमूद केल्याप्रमाणे, आपण आमचे AI एजंट तयार करण्यासाठी आणि संयोजित करण्यासाठी [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) वापरणार आहोत.

Microsoft Agent Framework आणि इतर आवश्यक पॅकेजेस इन्स्टॉल करण्यासाठी, या प्रोजेक्टच्या मूळ डिरेक्टरीमध्ये खालील कमांड चालवा:

```bash
pip install -r requirements.txt
```

### .env व्हेरिएबल्स सेटअप करा

या कोर्समधील कोड नमुने चालवण्यासाठी, आपण या प्रोजेक्टच्या मूळ डिरेक्टरीमध्ये `.env` फाइल तयार करणे आवश्यक आहे.

सोप्या साठी, आपण दिलेली `.env.example` फाइल कॉपी करू शकता:

```bash
cp .env.example .env
```
