## 🌱 Első lépések

Ez a tanfolyam leckéket tartalmaz, amelyek az AI-ügynökök építésének és telepítésének alapjait fedik le.

Minden lecke az előzőre épül, ezért javasoljuk, hogy az elejéről kezdve haladj végig.

Ha többet szeretnél felfedezni az AI-ügynök témákról, megtekintheted az [AI-ügynökök kezdőknek tanfolyamot](https://aka.ms/ai-agents-beginners).

### Találkozz más tanulókkal, kapj választ a kérdéseidre

Ha elakadsz, vagy bármilyen kérdésed van az AI-ügynökök építésével kapcsolatban, csatlakozz dedikált Discord csatornánkhoz a [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) oldalon.

### Amire szükséged van

Minden leckének van saját kódpéldája, amelyet helyben futtathatsz. Ezt a repót fork-olhatod, hogy létrehozd a saját példányodat: [fork this repo](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork).

A tanfolyam jelenleg a következőket használja:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

Kérjük, győződj meg róla, hogy hozzáférsz ezekhez a szolgáltatásokhoz, mielőtt elkezdenéd.

További lehetőségek a modell hosztolásával és szolgáltatásokkal kapcsolatban hamarosan érkeznek.
