## 🌱 စတင်ခြင်း

ဤသင်တန်းတွင် AI Agent များ တည်ဆောက်ခြင်းနှင့် တာဝန်ပေးခြင်း၏ အခြေခံများ အကြောင်းသင်ကြားပေးသော အခန်းများ ပါဝင်သည်။

ပေါင်းစပ်ထားသော အခန်းများသည် ယခင်အခန်းတွင် အခြေခံ၍ တည်ဆောက်ထားသောကြောင့် စတင်အမှတ်အသားမှ အဆုံးအထိ အဆင့်လိုက် သင်ကြားရန် အကြံပြုသည်။

AI Agent သင်ခန်းစာများ ပိုမိုလေ့လာလိုပါက [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) ကို ကြည့်ရှုနိုင်ပါသည်။

### အခြား သင်ယူသူများနှင့် တွေ့ဆုံခြင်း၊ မေးခွန်းများ ဖြေရှင်းခြင်း

AI Agent တည်ဆောက်ခြင်းနှင့်ပတ်သက်၍ စိတ်ပျက်ကွယ်သွားပါက သင်၏မေးခွန်းများ ဖော်ပြလိုပါက [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) တွင် ကြည့်ရှုသူများဆီသို့ ဝင်ရောက်မေးမြန်းနိုင်ပါသည်။

### လိုအပ်သော အရာများ

အခန်းတိုင်းတွင် ကိုယ်ပိုင် code နမူနာ တစ်ခုစီ ပါဝင်ပြီး ဒေသတွင်းတွင် အသုံးပြုနိုင်ပါသည်။ ကိုယ်ပိုင်ကူးယူရန် [repo ကို fork လုပ်ပါ](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)။

ယခုသင်တန်းတွင် အောက်ပါ အရာများကို အသုံးပြုပါသည် -

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

စတင်ရန် မတိုင်ခင် ဤ၀န်ဆောင်မှုများကို ဝင်ရောက်အသုံးပြုနိုင်မှုရှိသည်ကို သေချာစေရန် လိုအပ်ပါသည်။

မကြာမီ ဦးဆောင် မော်ဒယ်ပိုင်ရှင်နှင့် ဝန်ဆောင်မှုများ ပိုမိုထပ်မံလာမည်ဖြစ်သည်။

## ဆောင်ရွက်ခြင်း

ဤပရောဂျက်သည် ပံ့ပိုးမှုများနှင့် အကြံပြုချက်များကို ကြိုဆိုပါသည်။ ပံ့ပိုးမှုအများစုမှာ သင်အောက်ပါအချက်ကို သဘောတူဝင်ရန် လိုအပ်ပါသည် - သင်၏ ပံ့ပိုးမှုကို အသုံးပြုခွင့်အတွက် ခွင့်ပြုချက်ရှိပြီး ဖြန့်ဝေခွင့်ကို အသင့်ရှိကြောင်း သတင်းပို့ရန် Contributor License Agreement (CLA) တစ်ခုကို သဘောတူကြောင်း ပြောကြားရမည်။ အသေးစိတ်များအတွက် <https://cla.opensource.microsoft.com> ကို ကြည့်ရန်။

သင် pull request တင်သည့်အခါတွင် CLA bot တစ်ခုသည် သင် CLA တစ်ခု ပေးရန် လိုအပ်မလိုအပ်ကို အလိုအလျောက် ဆုံးဖြတ်နိုင်ပြီး pull request ကို ကောင်းစွာ အမှတ်အသားဖြင့် တင်ပြပေးပါမည် (ဥပမာ- အခြေအနေစစ်ဆေးမှု၊ မှတ်ချက်)။ bot ကောင်းစွာ ပေးသည့် အညွှန်းများကိုသာ လိုက်နာပါ။ မည်သည့် repo မဆို အတူတူ CLA ကို တစ်ကြိမ်သာ လုပ်ဆောင်ရမည် ဖြစ်သည်။

ဤပရောဂျက်သည် [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံ လိုက်နာထားပါသည်။
ပိုမိုသိရှိလိုပါက [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) ကို ကြည့်ရှုပါ သို့မဟုတ် မေးခွန်းများ ၊ မှတ်ချက်များအတွက် [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်နိုင်ပါသည်။

## ကုန်အမှတ်တံဆိပ်များ

ဤပရောဂျက်တွင် ပရောဂျက်များ၊ ထုတ်ကုန်များ သို့မဟုတ် ဝန်ဆောင်မှုများအတွက် ကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်များ ပါရှိနိုင်သည်။ Microsoft ၏ ကုန်အမှတ်တံဆိပ် သို့မဟုတ် အမှတ်တံဆိပ်များကို တရားဝင်အသုံးပြုခြင်းသည် [Microsoft ၏ ကုန်အမှတ်တံဆိပ်နှင့် Brand လမ်းညွှန်များ](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general) အောက်ရှိ စည်းမျဉ်းများနဲ့ နောက်လိုက်တွဲရန် ဖြစ်သည်။ ဤပရောဂျက်၏ ပြင်ဆင်ထားသော ဗားရှင်းများတွင် Microsoft ၏ ကုန်အမှတ်တံဆိပ် သို့မဟုတ် အမှတ်တံဆိပ်များကို အသုံးပြုမှုသည် ရှုပ်ထွေးမှုမဖြစ်စေရန် သည်းခံရမည် ဖြစ်ပြီး Microsoft အား တာဝန်ယူမှုပြုထားသည့်အတိုင်း မမြင်တွေ့ဖြစ်သွားစေရန် ဖြစ်သည်။ တတိယပါတီကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်များကို အသုံးပြုမည်ဆိုပါက ထိုတတိယပါတီ၏ မူဝါဒများကိုလိုက်နာရမည်ဖြစ်သည်။
