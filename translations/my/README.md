## 🌱 စတင်သင်ကြားခြင်း

ဒီသင်တန်းမှာ AI Agents ဖန်တီးခြင်းနှင့် ထည့်သွင်းခြင်း အခြေခံများကို သင်ကြားပေးပါတယ်။

တိုင်းသင်ခန်းစာသည် ယခင်သင်ခန်းစာပေါ်တွင် တည်ဆောက်ထားသည့်အတွက် အစမှ စတင်ပြီး အဆုံးအထိ အဆင့်ဆင့် မည်သို့ ကြည့်နေကြဖို့ အကြံပြုပါတယ်။

AI Agent ကဏ္ဍနှင့် ပိုမိုလေ့လာလိုလျှင် [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) ကို ကြည့်ရှုနိုင်ပါတယ်။

### အခြား သင်တန်းသားများနှင့် တွေ့ဆုံမေးမြန်းခြင်း

AI Agent ဖန်တီးရာတွင် အခက်အခဲရှိပါက သို့မဟုတ် မေးခွန်းများရှိပါက [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) တွင်ရှိသည့် dedicated Discord Channel မှာ လာရောက် ဆွေးနွေးနိုင်ပါတယ်။

### လိုအပ်သော အရာများ

တိုင်းသင်ခန်းစာမှာ ကိုယ်ပိုင် local တွင် ပြေးနိုင်သောကုတ်နမူနာ ပါပါတယ်။ ကိုယ်ပိုင် စိတ်ကြိုက် မိတ္တူ ဖန်တီးရန် [repo ကို fork လုပ်ပါ](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)။

ယခုသင်တန်းတွင် အသုံးပြုနေသည်မှာ –

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

စတင်မလုပ်မီ အထက်ဖော်ပြခဲ့သည့် ဝန်ဆောင်မှုများ သုံးနိုင်မှုရှိမရှိ စစ်ဆေးပါ။

အနာဂတ်တွင် မော်ဒယ်တင်ခြင်းနှင့် ဝန်ဆောင်မှုဆိုင်ရာ ရွေးချယ်စရာ နောက်ထပ်လမ်းများ ထည့်သွင်းစီစဉ်နေဆဲ ဖြစ်ပါတယ်။

ဒီပရောဂျက်တွင် ပရောဂျက်များ၊ ထုတ်ကုန်များ သို့မဟုတ် ဝန်ဆောင်မှုများအတွက် ကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်ပုံများ ပါဝင်နိုင်သည်။ Microsoft ကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်ပုံများကို သုံးစွဲခြင်းသည် [Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general) အရ စည်းကမ်းများနှင့် ကိုက်ညီရမည်။ ဒီပရောဂျက်တွင် Microsoft ကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်ပုံများ ပြင်ဆင်ထားသော ဗားရှင်းများအား အသုံးပြုလိုက်လျှင် Microsoft ထောက်ပံ့မှု ရှိကြောင်း သို့မဟုတ် ရောထွေးမှု ဖြစ်စေရန် မလုပ်သင့်ပါ။ တတိယပါတီ ကုန်အမှတ်တံဆိပ်များ သို့မဟုတ် အမှတ်တံဆိပ်ပုံများကို အသုံးပြုခြင်းသည် အဆိုပါ တတိယပါတီ မူဝါဒများနှင့် ကိုက်ညီရမည်ဖြစ်သည်။
