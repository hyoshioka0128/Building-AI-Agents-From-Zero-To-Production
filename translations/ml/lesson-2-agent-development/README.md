# പാഠം 2 ഏജന്റ് വികസനം

"ബിൽഡിംഗ് എഐ ഏജന്റ് ഫ്രം സീറോ ടു പ്രൊഡക്ഷൻ കോഴ്‌സ്"യുടെ രണ്ടാം പാഠത്തിലേക്ക് സ്വാഗതം!

ഈ പാഠത്തിൽ നാം പരിചയപ്പെടുന്നത്:

- ഞങ്ങളുടെ എഐ ഏജന്റുകൾ സൃഷ്‌ടിക്കാനുള്ള ഉപകരണങ്ങൾ

- വികസന വിഭവങ്ങളുടെ സജ്ജീകരണ നിർദ്ദേശങ്ങൾ

- എഐ ഏജന്റ് വികസന മികച്ച പ്രക്രിയകൾ

- എഐ ഏജന്റുകൾ സൃഷ്‌ടിക്കുന്നതിന് കോഡ് വാക്ക്‌ത്രൂ

നമുക്ക് ആദ്യം നാം എഐ ഏജന്റുകൾ സൃഷ്‌ടിക്കാൻ ഉപയോഗിക്കുന്ന ഉപകരണങ്ങൾ നോക്കാം.

## ഉപകരണങ്ങളും സജ്ജീകരണ നിർദ്ദേശങ്ങളും

### മൈക്രോസോഫ്റ്റ് ഫൗൻഡ്രി

വലുതായ ഭാഷാ മോഡലുകൾ (LLMs) ലഭ്യമാകുന്നതിനായി നാം [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ഉപയോഗിക്കുമെന്നും അറിയാം. ഫൗൻഡ്രി ഉപയോഗിക്കാൻ ചില ചെലവുകൾ ഉണ്ടാകുന്നതിനാൽ, ഇതിനായി അക്കൗണ്ട് സജ്ജമാക്കുന്നതിനുള്ള നിർദ്ദേശങ്ങൾ പാലിക്കുക, ഇപ്പോൾ നിങ്ങളുടെ അക്കൗണ്ട് ഇല്ലെങ്കിൽ.

### OpenAI മോഡലുകൾ

ഈ കോഴ്സിലെ ഏജന്റ് കോഡ് സാമ്പിളുകൾ [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) വഴി OpenAI മോഡലുകൾ ഉപയോഗിക്കുന്നതായി സജ്ജീകരിച്ചിരിക്കുന്നു.

ഫൗൻഡ്രി ഉപയോഗിച്ച് ഒരു മോഡൽ എങ്ങനെ വിന്യാസം ചെയ്യാമെന്ന് മനസിലാക്കാൻ ഈ ഗൈഡ് ഉപയോഗിക്കുക: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

ഈ കോഴ്‌സിനു GPT-4.1 അല്ലെങ്കിൽ അതിനു ശേഷം വന്ന മോഡൽ തിരഞ്ഞെടുക്കുക.

### Microsoft Agent Framework

മുമ്പ് പറഞ്ഞതുപോലെ, ഞങ്ങൾ [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ഉപയോഗിച്ച് എഐ ഏജന്റുകൾ സൃഷ്‌ടിക്കുകയും ഏകോപിപ്പിക്കുകയും ചെയ്യും.

Microsoft Agent Frameworkയും മറ്റ് ആവശ്യമായ പാക്കേജുകളും ഇൻസ്റ്റാൾ ചെയ്യാൻ, ഈ പ്രോജക്ടിന്റെ റൂട്ട് ഡയറക്ടറിയിലാണ് താഴെയുള്ള കമാൻഡ് പ്രവർത്തിപ്പിക്കുക:

```bash
pip install -r requirements.txt
```

### .env ചൂണ്ടികൾ സജ്ജീകരിക്കൽ

ഈ കോഴ്‌സിലെ കോഡ് സാമ്പിളുകൾ പ്രവർത്തിപ്പിക്കാൻ, നിങ്ങൾ ഈ പ്രോജക്ടിന്റെ റൂട്ട് ഡയറക്ടറിയിൽ `.env` ഫയൽ സൃഷ്ടിക്കേണ്ടതുണ്ട്.

ഇത് എളുപ്പമാക്കാൻ, നീങ്ങളുടെ ഉപയോഗങ്ങൾക്ക് നൽകിയ `.env.example` ഫയൽ കാപി ചെയ്യാം:

```bash
cp .env.example .env
```
