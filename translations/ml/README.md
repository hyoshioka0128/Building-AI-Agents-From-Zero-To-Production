## 🌱 ആരംഭം

ഈ കോഴ്‌സിൽ എഐ ഏജന്റുകൾ സൃഷ്ടിക്കുകയും വിന്യസം ചെയ്യുകയും ചെയ്യുന്നതിനുള്ള അടിസ്ഥാന പാഠങ്ങൾ ഉൾപ്പടുന്നു.

ഓരോ പാഠവും മുൻപത്തെ പാഠത്തെ അടിസ്ഥാനമാക്കിയുള്ളത് ആകുന്നതിനാൽ തുടക്കം മുതൽ തുടർന്നു അവസാനത്തേക്കു വരെ പഠിക്കാൻ സൈഫ്‌ഷ്യമാണ്.

നിങ്ങൾക്ക് എഐ ഏജന്റ് വിഷങ്ങൾ കുറിച്ച് കൂടുതൽ അറിയണമെങ്കിൽ [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) പരിശോധിക്കാം.

### മറ്റ് പഠിതാക്കളെ കാണൂ, നിങ്ങളുടെ ചോദ്യങ്ങൾക്ക് ഉത്തരം ലഭിക്കൂ

പഠനത്തിൽ കുടുങ്ങിയാൽ അല്ലെങ്കിൽ എഐ ഏജന്റുകൾ നിർമ്മാണത്തെക്കുറിച്ച് എന്തെങ്കിലും ചോദ്യങ്ങൾ ഉണ്ടെങ്കിൽ, [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) -ലെ നമുക്ക് സമർപ്പിച്ചിരിക്കുന്ന ഡിസ്കോർഡ് ചാനലിൽ ചേർന്നു ചർച്ച ചെയ്യാം.

### നിങ്ങൾക്ക് ആവശ്യമായതു

ഓരോ പാഠത്തിനും തങ്ങളിലാണ് കോഡ് സാമ്പിൾ നൽകപ്പെട്ടിരിക്കുന്നത്, അത് പ്രാദേശികമായി പ്രവർത്തിപ്പിക്കാം. നിങ്ങൾക്ക് [ഈ റിപോ ഫോർക്കു ചെയ്യാം](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) സ്വന്തം കോപ്പി സൃഷ്ടിക്കാൻ.

ഈ കോഴ്‌സ് ഇപ്പോൾ താഴെപറയുന്നവ ഉപയോഗിക്കുന്നു:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

തുടങ്ങുന്നതിനും മുമ്പ് ഈ സേവനങ്ങൾക്ക് പ്രവേശനം ഉറപ്പാക്കുക.

മോഡൽ ഹോസ്റ്റിംഗ്, സേവനങ്ങൾ സംബന്ധിച്ചുള്ള കൂടുതൽ ഓപ്ഷനുകൾ ഉടൻ വരും.

നിങ്ങൾ ഒരു പുൾ അപേക്ഷ സമർപ്പിക്കുമ്പോൾ, CLA ബോട്ട് സ്വയം നിങ്ങൾക്ക് CLA നൽകേണ്ടതുണ്ടോ എന്ന് നിർണ്ണയിച്ച് PR യെ അനുയോജ്യമായി അലങ്കരിച്ചു (ഉദാ: സ്ഥിതിവിവര പരിശോധന, കമന്റ്) നൽകും. ബോട്ടിന്റെ നിർദ്ദേശങ്ങൾ പാലിക്കുക. ഞങ്ങളുടെ CLA ഉപയോഗിക്കുന്ന എല്ലാ മേഖലകളിലും നിങ്ങൾക്ക് ഇത് ഒരിക്കൽ മാത്രം ചെയ്യേണ്ടതുണ്ട്.

ഈ പ്രോജക്റ്റ് [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) സ്വീകരിച്ചിട്ടുണ്ട്. കൂടുതൽ വിവരങ്ങൾക്ക് [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) കാണുക അല്ലെങ്കിൽ [opencode@microsoft.com](mailto:opencode@microsoft.com) എന്നതിൽ ഏതു അധിക ചോദ്യങ്ങളോ അഭിപ്രായങ്ങളോ ഉണ്ടെങ്കിൽ ബന്ധപ്പെടുക.

## ട്രേഡ്മാർക്കുകൾ

ഈ പ്രോജക്റ്റിൽ പ്രോജക്റ്റുകൾ, ഉൽപ്പന്നങ്ങൾ, അല്ലെങ്കിൽ സേവനങ്ങൾ സംബന്ധിച്ച ട്രേഡ്മാർക്കുകൾ അല്ലെങ്കിൽ ലോഗോകൾ ഉണ്ടായിരിക്കും. Microsoft ട്രേഡ്മാർക്കുകൾ അല്ലെങ്കിൽ ലോഗോകൾ ഉപയോഗിക്കുന്നത് [Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general) അനുസരിച്ച് അനുമതിയുള്ളതും പാലിക്കേണ്ടതുമാണ്. ഈ പ്രോജക്റ്റിന്റെ തിരുത്തിയ പതിപ്പുകളിൽ Microsoft ട്രേഡ്മാർക്കുകൾ അല്ലെങ്കിൽ ലോഗോകൾ ഉപയോഗിക്കുന്നത് വാദമുണ്ടാക്കാനോ അല്ലെങ്കിൽ Microsoft മേധാവിത്തം സൂചനയുണ്ടാക്കാനോ വേണ്ട. മൂന്നാം കക്ഷി ട്രേഡ്മാർക്കുകൾ അല്ലെങ്കിൽ ലോഗോകൾ ഉപയോഗിക്കുന്നത് ഇവരുടെ നയം അനുസരിച്ചിരിക്കണം.
