## 🌱 開始

本課程包含關於建立與部署 AI 代理人基礎的課程。

每堂課都建立在前一堂的基礎上，因此建議從頭開始，依序完成所有課程。

如果你想進一步探索 AI 代理人主題，可以參考 [AI 代理人初學者課程](https://aka.ms/ai-agents-beginners).

### 與其他學習者交流，獲得你的問題解答

如果你卡住或對建立 AI 代理人有任何問題，加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 的專用 Discord 頻道。

### 所需項目

每堂課都有可在本機執行的範例程式碼。你可以 [fork 此儲存庫](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) 來建立自己的副本。

本課程目前使用下列項目：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

請在開始之前確保你能存取這些服務。

有關模型託管與服務的更多選項即將推出。 

## 🗃️ 課程

| **課程**         | **描述**                                                                                  |
|--------------------|--------------------------------------------------------------------------------------------------|
| [代理人設計](./lesson-1-agent-design/README.md)       | 介紹我們的「開發者入職」代理用例，以及如何設計有效的代理人  |
| [代理人開發](./lesson-2-agent-development/README.md)  | 使用 Microsoft Agent Framework (MAF)，建立三個代理人以協助新開發者入職。       |
| [代理人評估](./lesson-3-agent-evals/README.md)  | 使用 Microsoft Foundry，瞭解我們的 AI 代理人的效能以及如何改進。 |
| [代理人部署](./lesson-4-agent-deployment/README.md)   | 使用 Hosted Agents 和 OpenAI Chatkit，了解如何將 AI 代理人部署到生產環境。       |
