# పాఠం 2 ఏజెంట్ అభివృద్ధి

"శూన్య నుంచి ఉత్పత్తి వరకు AI ఏజెంట్ నిర్మించటం కోర్సు" రెండవ పాఠానికి స్వాగతం!

ఈ పాఠంలో మేము కవర్ చేయబోతున్నాము:

- మన AI ఏజెంట్లను సృష్టించడానికి ఉపయోగించే టూల్స్
  
- మా అభివృద్ధి వనరుల సెటప్ సూచనలు

- AI ఏజెంట్ అభివృద్ధి ఉత్తమ పద్ధతులు
  
- మన AI ఏజెంట్లను సృష్టించేందుకు కోడ్ వాక్‌థ్రూ
  
మన AI ఏజెంట్లను సృష్టించడానికి ఉపయోగించే టూల్స్‌ను చూస్తూ ప్రారంభిద్దాం.

## టూల్స్ మరియు సెటప్ సూచనలు

### మైక్రోసాఫ్ట్ ఫౌండ్రీ

లార్జ్ లాంగ్వేజ్ మోడల్స్ (LLMs) అపరిక్షేపానికి మేము [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ను ఉపయోగించబోతున్నాము. ఫౌండ్రీ ఉపయోగించే ఖర్చులు ఉన్నట్లు గుర్తుంచుకోండి కాబట్టి ఖాతా సెటప్ కోసం సూచనలను అనుసరించండి, మీకు ఇప్పటికే యాక్సెస్ లేకపోతే.

### OpenAI మోడల్స్

ఈ కోర్సులో ఏజెంట్ కోడ్ ఉదాహరణలు [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ద్వారా OpenAI మోడల్స్ ఉపయోగించే విధంగా సెటప్ చేయబడ్డాయి.

ఫౌండ్రీ ఉపయోగించి మోడల్‌ను డిప్లాయ్ చేయడం ఎలా అనేదాన్ని తెలుసుకోవడానికి ఈ గైడ్‌ను ఉపయోగించండి: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

ఈ కోర్సు కోసం GPT-4.1 లేదా తర్వాత మోడల్ లో ఒకదాన్ని ఎంచుకోండి.

### మైక్రోసాఫ్ట్ ఏజెంట్ ఫ్రేమ్‌వర్క్

ముందుగానే చెప్పినట్లుగా, మేము AI ఏజెంట్లను సృష్టించడానికి మరియు ఒార్కిస్ట్రేట్ చేయడానికి [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ను ఉపయోగించబోతున్నాము.

Microsoft Agent Framework మరియు ఇతర అవసరమైన ప్యాకేజీలను ఇన్‌స్టాల్ చేయడానికి, ఈ ప్రాజెక్ట్ యొక్క రూట్ డైరెక్టరీలో ఈ కమాండ్‌ను నడపండి:

```bash
pip install -r requirements.txt
```

### .env వేరియబుల్స్ సెటప్ చేయండి

ఈ కోర్సులో కోడ్ ఉదాహరణలు నడపడానికి, ఈ ప్రాజెక్ట్ యొక్క రూట్ డైరెక్టరీలో ఒక `.env` ఫైల్ సృష్టించుకోవాలి.

దీనిని సులభతరం చేసుకోవడానికి, మీరు అందించిన `.env.example` ఫైల్‌ని కాపీ చేయవచ్చు:

```bash
cp .env.example .env
```
