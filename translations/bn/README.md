## 🌱 শুরু করুন

এই কোর্সটি এআই এজেন্ট তৈরি ও স্থাপনের মূল বিষয়গুলি কভার করে।

প্রতিটি পাঠের ওপর পূর্ববর্তী পাঠের ভিত্তি করে তৈরি, তাই আমরা পরামর্শ দিই শুরু থেকে শুরু করে শেষ পর্যন্ত ধারাবাহিকভাবে এগোনোর।

আপনি যদি এআই এজেন্ট বিষয়ক আরও অনুসন্ধান করতে চান, তবে আপনি [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) দেখতে পারেন।

### অন্যান্য শিক্ষার্থীদের সাথে পরিচিত হন, আপনার প্রশ্নের উত্তর পান

যদি আপনি আটকে যান অথবা এআই এজেন্ট তৈরির বিষয়ে কোনো প্রশ্ন থাকে, আমাদের নিবেদিত ডিসকর্ড চ্যানেলে যোগ দিন [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6)।

### যা আপনাকে প্রয়োজন

প্রত্যেক পাঠের জন্য আলাদা কোড স্যাম্পল রয়েছে যা আপনি স্থানীয়ভাবে চালাতে পারবেন। আপনার নিজের কপি তৈরির জন্য আপনি [এই রিপো ফরক করতে পারেন](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)।

এই কোর্স বর্তমানে নিচেরগুলো ব্যবহার করে:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

শুরু করার আগে নিশ্চিত করুন যে আপনার কাছে এই সেবাগুলোর অ্যাক্সেস আছে।

মডেল হোস্টিং এবং সেবাগুলো নিয়ে আরও বিকল্প শীঘ্রই আসছে।

ট্রেডমার্ক বা লোগোর অনুমোদিত ব্যবহার নির্ভর করে এবং অবশ্যই অনুসরণ করতে হবে
[Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general)।
এই প্রকল্পের সংশোধিত সংস্করণে মাইক্রোসফট ট্রেডমার্ক বা লোগো ব্যবহার মাইক্রোসফট স্পনসরশিপ বা বিভ্রান্তি সৃষ্টি করা যাবেনা।
তৃতীয় পক্ষের ট্রেডমার্ক বা লোগোর কোন ব্যবহার ঐ পক্ষের নীতিমালা অনুসারে হবে।
