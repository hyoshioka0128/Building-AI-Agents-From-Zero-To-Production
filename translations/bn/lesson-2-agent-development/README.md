# Lesson 2 Agent Development

"শূন্য থেকে উৎপাদনে এআই এজেন্ট নির্মাণ কোর্স"-এর দ্বিতীয় পাঠে আপনাকে স্বাগতম!

এই পাঠে আমরা покриব:

- আমাদের AI এজেন্ট তৈরি করার জন্য টুলস
  
- আমাদের উন্নয়ন সংস্থানগুলোর সেটআপ নির্দেশিকা

- AI এজেন্ট উন্নয়নের সেরা অনুশীলন
  
- আমাদের AI এজেন্ট তৈরি করার জন্য কোড ওয়াকথ্রু
  
এখন চলুন দেখি আমরা কোন টুলস ব্যবহার করব আমাদের AI এজেন্ট তৈরি করতে।

## টুলস এবং সেটআপ নির্দেশিকা

### Microsoft Foundry

বড় ভাষা মডেল (LLMs) অ্যাক্সেসের জন্য আমরা ব্যবহার করব [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)। Foundry ব্যবহার করার জন্য ঝরিপাত খরচ আছে, তাই যদি আপনার অ্যাক্সেস না থাকে, অ্যাকাউন্ট সেটআপের নির্দেশাবলী অবশ্যই অনুসরণ করুন।

### OpenAI Models

এই কোর্সের এজেন্ট কোডের স্যাম্পলগুলো OpenAI মডেল Microsoft Foundry ব্যবহার করে সেটআপ করা হয়েছে।

Foundry ব্যবহার করে কিভাবে মডেল ডিপ্লয় করবেন এই গাইডটি ব্যবহার করুন: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

এই কোর্সের জন্য GPT-4.1 বা পরবর্তী যে কোনো একটি মডেল নির্বাচন করুন।

### Microsoft Agent Framework

আগে উল্লেখ করা হয়েছে, আমরা AI এজেন্ট তৈরি এবং সমন্বয়ের জন্য [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ব্যবহার করব।

Microsoft Agent Framework এবং অন্যান্য প্রয়োজনীয় প্যাকেজ ইনস্টল করতে, এই প্রকল্পের রুট ডিরেক্টরিতে নিচের কমান্ডটি চালান:

```bash
pip install -r requirements.txt
```

### .env ভেরিয়েবল সেটআপ

এই কোর্সের কোড স্যাম্পলগুলি চালানোর জন্য, এই প্রকল্পের রুট ডিরেক্টরিতে একটি `.env` ফাইল তৈরি করতে হবে।

সহজ করার জন্য, আপনি সরবরাহকৃত `.env.example` ফাইলটি কপি করতে পারেন:

```bash
cp .env.example .env
```
