# पाठ २ एजेन्ट विकास

"शून्यबाट उत्पादनसम्म एआई एजेन्ट बनाउने कोर्स" को दोस्रो पाठमा स्वागत छ!

यस पाठमा हामीले समेट्ने छौं:

- हाम्रा एआई एजेन्टहरू बनाउनका लागि उपकरणहरू
  
- विकास स्रोतहरू सेटअप गर्ने निर्देशनहरू

- एआई एजेन्ट विकासका उत्कृष्ट अभ्यासहरू
  
- हाम्रा एआई एजेन्टहरू बनाउनको लागि कोड यात्रा
  
हामीले एआई एजेन्टहरू बनाउन प्रयोग गर्ने उपकरणहरू हेरेर सुरु गरौं।

## उपकरण र सेटअप निर्देशनहरू

### Microsoft Foundry

ठूला भाषा मोडेलहरू (LLMs) पहुँचका लागि हामी [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) प्रयोग गर्नेछौं। Foundry प्रयोग गर्न केही लागतहरू लाग्छन् त्यसैले यदि तपाईंलाई अझै पहुँच छैन भने कृपया खाता सेटअप गर्ने निर्देशनहरू पालना गर्नुहोस्।

### OpenAI मोडेलहरू

यस कोर्सका एजेन्ट कोड नमूनाहरू OpenAI मोडेलहरू Microsoft Foundry मार्फत प्रयोग गर्न सेटअप भएका छन्।

Foundry प्रयोग गरेर मोडेल कसरी परिनियोजन गर्ने सिक्न यो गाइड उपयोग गर्नुहोस्: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

यस कोर्सका लागि GPT-4.1 वा पछि को कुनै एक मोडेल छान्नुहोस्।

### Microsoft Agent Framework

पहिले बताएझैं, हामीले एआई एजेन्टहरू सिर्जना र समन्वय गर्नका लागि [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) प्रयोग गर्नेछौं।

Microsoft Agent Framework र अन्य आवश्यक प्याकेजहरू इन्स्टल गर्न, यस परियोजनाको रुट डिरेक्टरीमा हुँदा तलको आदेश चलाउनुहोस्:

```bash
pip install -r requirements.txt
```

### .env भेरिएबलहरू सेटअप गर्नुहोस्

यस कोर्सका कोड नमूनाहरू चलाउन, तपाईंले यो परियोजनाको रुट डिरेक्टरीमा `.env` फाइल बनाउनुपर्नेछ।

सरल बनाउनको लागि, तपाईं उपलब्ध `.env.example` फाइल कपी गर्न सक्नुहुन्छ:

```bash
cp .env.example .env
```
