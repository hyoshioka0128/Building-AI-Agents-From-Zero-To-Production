# មេរៀនទី ២ ការអភិវឌ្ឍភ្នាក់ងារ

សូមស្វាគមន៍មកកាន់មេរៀនទីពីរនៃវគ្គសិក្សា "ការបង្កើតភ្នាក់ងារ AI ពីសូន្យទៅផលិតកម្ម"!

ក្នុងមេរៀននេះ យើងនឹងគ្របដណ្តប់៖

- ឧបករណ៍សម្រាប់បង្កើតភ្នាក់ងារ AI របស់យើង
  
- សេចក្តីណែនាំលំអិតសម្រាប់ធនធានអភិវឌ្ឍន៍របស់យើង

- អនុវត្តអំណោយភាពល្អបំផុតក្នុងការអភិវឌ្ឍភ្នាក់ងារ AI
  
- ការដើរយល់កូដសម្រាប់បង្កើតភ្នាក់ងារ AI របស់យើង
  
ចាប់ផ្តើមដោយពិនិត្យមើលឧបករណ៍ដែលយើងនឹងប្រើសម្រាប់បង្កើតភ្នាក់ងារ AI របស់យើង។

## ឧបករណ៍ និង សេចក្តីណែនាំដំឡើង

### Microsoft Foundry

សម្រាប់ការចូលដំណើរការម៉ូដែលភាសាធំ (LLMs) យើងនឹងប្រើ [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)។ មានថ្លៃដើមចំណាយដែលពាក់ព័ន្ធនឹងការប្រើប្រាស់ Foundry ដូច្នេះសូមប្រាកដថាបង្កើតគណនីតាមរយៈសេចក្តីណែនាំប្រសិនបើអ្នកមិនមានចូលដំណើរការមុនទេ។

### ម៉ូដែល OpenAI

ឧទាហរណ៍កូដភ្នាក់ងារនៅក្នុងវគ្គនេះត្រូវបានដំឡើងប្រើម៉ូដែល OpenAI តាមរយៈ [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)។

ប្រើមគ្គុទេសក៍នេះដើម្បីរៀនពីរបៀបដំឡើងម៉ូដែលដោយប្រើ Foundry ៖ [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

ជ្រើសរើសមួយម៉ូដែល GPT-4.1 ឬក្រោយសម្រាប់វគ្គនេះ។

### Microsoft Agent Framework

ដូចដែលបានរៀបរាប់មុននេះ យើងនឹងប្រើ [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ដើម្បីបង្កើត និងរៀបចំភ្នាក់ងារ AI របស់យើង។

ដើម្បីដំឡើង Microsoft Agent Framework និងកញ្ចប់ផ្សេងទៀតដែលត្រូវការ ធ្វើការបញ្ជា្រងខាងក្រោមនៅក្នុងថតរុក្ខជាតិគម្រោងនេះ៖

```bash
pip install -r requirements.txt
```

### ដំឡើងអថេរ .env

ដើម្បីដំណើរការឧទាហរណ៍កូដនៅក្នុងវគ្គនេះ អ្នកត្រូវបង្កើតឯកសារ `.env` នៅក្នុងថតរុក្ខជាតិគម្រោងនេះ។

ដើម្បីធ្វើអោយងាយស្រួល អ្នកអាចចម្លងឯកសារ `.env.example` ដែលបានផ្តល់ឲ្យ៖

```bash
cp .env.example .env
```
