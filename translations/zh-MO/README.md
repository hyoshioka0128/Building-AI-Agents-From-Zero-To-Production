## 🌱 入門指南

本課程涵蓋建構與部署 AI 代理的基礎知識。

每堂課都建立在前一堂課之上，我們建議從頭開始，一步步學習直到完成。

如果你想深入探索 AI 代理相關主題，可以參考 [AI 代理入門課程](https://aka.ms/ai-agents-beginners)。

### 認識其他學員，獲得問題解答

如果你在建構 AI 代理時遇到困難或有疑問，歡迎加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 中的專屬 Discord 頻道。

### 需要準備什麼

每堂課都有獨立的程式碼範例，可以在本地執行。你可以 [fork 此倉庫](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) 來建立自己的副本。

本課程目前使用以下服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI 服務](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

請確保你在開始前能夠存取這些服務。

更多關於模型托管和服務的選項即將推出。

## 🗃️ 課程內容

| **課程**                 | **說明**                                                                                      |
|--------------------------|---------------------------------------------------------------------------------------------|
| [代理設計](./lesson-1-agent-design/README.md)       | 介紹我們的「開發者上手」代理使用案例以及如何設計有效的代理                                  |
| [代理開發](./lesson-2-agent-development/README.md)  | 使用 Microsoft Agent Framework (MAF) 建立三個協助新開發者上手的代理                            |
