# Уџбеник 2 Развој агента

Добродошли у други уџбеник курса „Прављење AI агента од почетка до производње“!

У овом уџбенику ћемо обрадити:

- Алате за креирање наших AI агената
  
- Упутства за подешавање наших ресурса за развој

- Најбоље праксе за развој AI агената
  
- Прошетку кроз код за креирање наших AI агената
  
Почнимо тако што ћемо погледати алате које ћемо користити за креирање наших AI агената.

## Алати и упутства за подешавање

### Microsoft Foundry

За приступ великим језичким моделима (LLM) користићемо [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry). Коришћење Foundry-а подразумева трошкове, па вас молимо да пратите упутства за подешавање налога ако већ немате приступ.

### OpenAI модели

Примери кода агената у овом курсу су подешени да користе OpenAI моделе преко [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry).

Користите овај водич да научите како да поставите модел користећи Foundry: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

За овај курс изаберите један GPT-4.1 или новији модел.

### Microsoft Agent Framework

Као што је раније поменуто, користићемо [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) за креирање и оркестрацију наших AI агената.

Да бисте инсталирали Microsoft Agent Framework и остале потребне пакете, покрените следећу команду у коренском директоријуму овог пројекта:

```bash
pip install -r requirements.txt
```

### Подешавање .env променљивих

Да бисте покренули примере кода из овог курса, потребно је да направите `.env` фајл у коренском директоријуму овог пројекта.

Да би вам било лакше, можете копирати обезбеђени `.env.example` фајл:

```bash
cp .env.example .env
```
