## 🌱 入門指南

本課程涵蓋構建和部署 AI 代理的基本課題。

每個課程都是建立在前一堂課的基礎上，建議從頭開始，逐步學習直到完成。

如果想進一步探索 AI 代理相關主題，可以參考 [AI Agents For Beginenrs Course](https://aka.ms/ai-agents-beginners)。

### 與其他學員互動，獲得問題解答

如果在構建 AI 代理過程中遇到困難或有任何疑問，歡迎加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 的專屬 Discord 頻道。

### 你需要準備的東西

每堂課都有對應的代碼範例，可在本地運行。你可以[分叉此倉庫](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)來創建自己的副本。

本課程目前使用以下內容：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI 服務](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

請確保在開始前有權限訪問這些服務。

更多模型託管和服務選項即將推出。
