## 🌱 開始入門

本課程涵蓋建立及部署 AI 代理的基礎知識。

每個課程都是在前一課基礎上建立，建議從頭開始逐步完成。

如果想深入探索 AI 代理相關主題，可以參考 [AI 代理新手課程](https://aka.ms/ai-agents-beginners)。

### 認識其他學習者、有問題就問

如果遇到困難或有關於建立 AI 代理的任何問題，歡迎加入我們在 [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) 的專屬頻道。

### 您需要什麼

每堂課都有對應的代碼範例，可以在本地運行。您可以[叉出本倉庫](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) 來建立自己的副本。

本課程目前使用以下項目：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI 服務](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

開始前請確保您有權限使用這些服務。

更多模型主機與服務選項即將推出。
