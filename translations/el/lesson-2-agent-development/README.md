# Μαθήμα 2 Ανάπτυξη Πράκτορα

Καλωσήρθατε στο δεύτερο μάθημα του "Μαθήματος Δημιουργίας Πράκτορα Τεχνητής Νοημοσύνης από το Μηδέν έως την Παραγωγή"!

Σε αυτό το μάθημα θα καλύψουμε:

- Τα Εργαλεία για τη Δημιουργία των Πρακτόρων ΤΝ μας
  
- Οδηγίες Ρυθμίσεων για τους Πόρους Ανάπτυξης μας

- Καλύτερες Πρακτικές Ανάπτυξης Πρακτόρων ΤΝ
  
- Ανάλυση Κώδικα για τη Δημιουργία των Πρακτόρων ΤΝ μας
  
Ας ξεκινήσουμε εξετάζοντας τα εργαλεία που θα χρησιμοποιήσουμε για να δημιουργήσουμε τους Πράκτορες ΤΝ μας.

## Εργαλεία και Οδηγίες Ρυθμίσεων

### Microsoft Foundry

Για πρόσβαση σε Μεγάλα Μοντέλα Γλώσσας (LLMs) θα χρησιμοποιούμε το [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry). Υπάρχουν κόστη που σχετίζονται με τη χρήση του Foundry, οπότε παρακαλώ βεβαιωθείτε ότι θα ακολουθήσετε τις οδηγίες για τη ρύθμιση λογαριασμού αν δεν έχετε ήδη πρόσβαση.

### Μοντέλα OpenAI

Τα παραδείγματα κώδικα πρακτόρων σε αυτό το μάθημα έχουν ρυθμιστεί για να χρησιμοποιούν μοντέλα OpenAI μέσω του [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry).

Χρησιμοποιήστε αυτόν τον οδηγό για να μάθετε πώς να αναπτύξετε ένα μοντέλο χρησιμοποιώντας το Foundry: [Ανάπτυξη Μοντέλων Microsoft Foundry στην πύλη Foundry](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

Επιλέξτε ένα μοντέλο GPT-4.1 ή νεότερο για αυτό το μάθημα.

### Microsoft Agent Framework

Όπως αναφέρθηκε προηγουμένως, θα χρησιμοποιήσουμε το [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) για τη δημιουργία και τον συντονισμό των Πρακτόρων ΤΝ μας.

Για να εγκαταστήσετε το Microsoft Agent Framework και άλλα απαιτούμενα πακέτα, εκτελέστε την ακόλουθη εντολή ενώ βρίσκεστε στον ριζικό κατάλογο αυτού του έργου:

```bash
pip install -r requirements.txt
```

### Ρυθμίσεις Μεταβλητών .env

Για να εκτελέσετε τα παραδείγματα κώδικα σε αυτό το μάθημα, θα χρειαστεί να δημιουργήσετε ένα αρχείο `.env` στον ριζικό κατάλογο αυτού του έργου.

Για να γίνει πιο εύκολο, μπορείτε να αντιγράψετε το παρεχόμενο αρχείο `.env.example`:

```bash
cp .env.example .env
```
