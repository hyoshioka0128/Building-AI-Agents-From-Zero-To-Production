## 🌱 Pour commencer

Ce cours comprend des leçons couvrant les fondamentaux de la construction et du déploiement d'agents IA.

Chaque leçon s'appuie sur la précédente, nous recommandons donc de commencer depuis le début et de progresser jusqu’à la fin.

Si vous souhaitez explorer davantage les sujets liés aux agents IA, vous pouvez consulter le [Cours pour débutants sur les agents IA](https://aka.ms/ai-agents-beginners).

### Rencontrez d'autres apprenants, obtenez des réponses à vos questions

Si vous êtes bloqué ou avez des questions sur la construction d'agents IA, rejoignez notre chaîne Discord dédiée dans le [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### Ce dont vous avez besoin

Chaque leçon a son propre exemple de code que vous pouvez exécuter localement. Vous pouvez [forker ce dépôt](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) pour créer votre propre copie.

Ce cours utilise actuellement les éléments suivants :

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

Veuillez vous assurer d'avoir accès à ces services avant de commencer.

D'autres options concernant l'hébergement de modèles et les services arrivent bientôt.

## 🗃️ Leçons

| **Leçon**          | **Description**                                                                                   |
|--------------------|----------------------------------------------------------------------------------------------------|
| [Conception d’agent](./lesson-1-agent-design/README.md)       | Une introduction à notre cas d’utilisation « Intégration développeur » et comment concevoir des agents efficaces  |
| [Développement d’agent](./lesson-2-agent-development/README.md)  | Utilisez Microsoft Agent Framework (MAF) pour créer 3 agents aidant les nouveaux développeurs dans leur intégration.       |
| [Évaluations d’agents](./lesson-3-agent-evals/README.md)  | Avec Microsoft Foundry, découvrez la performance de nos agents IA et comment les améliorer. |
| [Déploiement d’agent](./lesson-4-agent-deployment/README.md)   | Utilisez les agents hébergés et OpenAI Chatkit pour voir comment déployer un agent IA en production.       |
