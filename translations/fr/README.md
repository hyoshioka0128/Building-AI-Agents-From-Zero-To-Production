## 🌱 Pour commencer

Ce cours contient des leçons couvrant les fondamentaux de la construction et du déploiement d’agents IA.

Chaque leçon s’appuie sur la précédente, nous recommandons donc de commencer au début et d’avancer jusqu’à la fin.

Si vous souhaitez explorer davantage les sujets liés aux agents IA, vous pouvez consulter le [Cours Agents IA pour débutants](https://aka.ms/ai-agents-beginners).

### Rencontrez d’autres apprenants, obtenez des réponses à vos questions

Si vous êtes bloqué ou avez des questions sur la construction d’agents IA, rejoignez notre canal Discord dédié dans le [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### Ce dont vous avez besoin

Chaque leçon a son propre exemple de code que vous pouvez exécuter localement. Vous pouvez [forker ce dépôt](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) pour créer votre propre copie.

Ce cours utilise actuellement les éléments suivants :

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

Veuillez vous assurer que vous avez accès à ces services avant de commencer.

D’autres options autour de l’hébergement de modèles et des services arrivent bientôt.

## 🗃️ Leçons

| **Leçon**         | **Description**                                                                                  |
|--------------------|--------------------------------------------------------------------------------------------------|
| [Conception d’agent](./lesson-1-agent-design/README.md)       | Une introduction à notre cas d’usage « Onboarding développeur » et comment concevoir des agents efficaces  |
| [Développement d’agent](./lesson-2-agent-development/README.md)  | Utilisez le Microsoft Agent Framework (MAF) pour créer 3 agents aidant les nouveaux développeurs à s’intégrer.   |
| [Évaluations d’agent](./lesson-3-agent-evals/README.md)  | Avec Microsoft Foundry, découvrez les performances de nos agents IA et comment les améliorer.  |
| [Déploiement d’agent](./lesson-4-agent-deployment/README.md)   | Avec Hosted Agents et OpenAI Chatkit, découvrez comment déployer un agent IA en production.  |
