## 🌱 Memulakan

Kursus ini mempunyai pelajaran yang merangkumi asas-asas membina dan mengedarkan Ejen AI.

Setiap pelajaran dibina atas pelajaran sebelumnya, jadi kami mengesyorkan bermula dari awal dan teruskan sehingga akhir.

Jika anda ingin meneroka lebih lanjut tentang topik Ejen AI, anda boleh semak [Kursus Ejen AI untuk Pemula](https://aka.ms/ai-agents-beginners).

### Bertemu Pelajar Lain, Dapatkan Jawapan untuk Soalan Anda

Jika anda tersekat atau mempunyai sebarang soalan mengenai membina Ejen AI, sertai Saluran Discord khusus kami dalam [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### Apa yang Anda Perlukan

Setiap Pelajaran mempunyai sampel kod yang boleh anda jalankan secara lokal. Anda boleh [fork repo ini](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) untuk membuat salinan sendiri.

Kursus ini pada masa ini menggunakan:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

Sila pastikan anda mempunyai akses kepada perkhidmatan ini sebelum memulakan.

Pilihan lebih banyak tentang pengehosan model dan perkhidmatan akan datang tidak lama lagi.
