## 🌱 Memulai

Kursus ini memiliki pelajaran yang membahas dasar-dasar membangun dan menerapkan Agen AI.

Setiap pelajaran dibangun dari pelajaran sebelumnya, jadi kami menyarankan untuk memulai dari awal dan melanjutkan hingga akhir.

Jika Anda ingin menjelajahi lebih banyak tentang topik Agen AI, Anda dapat memeriksa [Kursus AI Agents Untuk Pemula](https://aka.ms/ai-agents-beginners).

### Bertemu Dengan Pelajar Lain, Dapatkan Jawaban atas Pertanyaan Anda

Jika Anda mengalami kesulitan atau memiliki pertanyaan tentang pembangunan Agen AI, bergabunglah dengan Saluran Discord khusus kami di [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### Apa Yang Anda Butuhkan

Setiap Pelajaran memiliki contoh kode sendiri yang dapat Anda jalankan secara lokal. Anda dapat [fork repo ini](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) untuk membuat salinan sendiri.

Kursus ini saat ini menggunakan hal-hal berikut:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

Pastikan Anda memiliki akses ke layanan ini sebelum memulai.

Pilihan lain terkait hosting model dan layanan akan segera hadir.

## 🗃️ Pelajaran

| **Pelajaran**             | **Deskripsi**                                                                                  |
|--------------------------|--------------------------------------------------------------------------------------------------|
| [Desain Agen](./lesson-1-agent-design/README.md)           | Pengenalan ke Kasus Penggunaan Agen "Developer Onboarding" kami dan cara mendesain agen yang efektif  |
| [Pengembangan Agen](./lesson-2-agent-development/README.md) | Menggunakan Microsoft Agent Framework (MAF), buat 3 agen untuk membantu pengembang baru onboarding.    |
| [Evaluasi Agen](./lesson-3-agent-evals/README.md)          | Menggunakan Microsoft Foundry, cari tahu seberapa baik kinerja Agen AI kami dan cara meningkatkannya.  |
| [Penerapan Agen](./lesson-4-agent-deployment/README.md)      | Menggunakan Hosted Agents dan OpenAI Chatkit, lihat cara menerapkan Agen AI ke produksi.                |
