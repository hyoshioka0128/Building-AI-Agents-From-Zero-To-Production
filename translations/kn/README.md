## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಮತ್ತು ನಿಯೋಜಿಸುವ ಮೂಲಭೂತಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.

ಪ್ರತಿಯೊಂದು ಪಾಠವು ಹಿಂದಿನ ಪಾಠದ ಮೇಲೆ ನಿರ್ಮಿಸಲಾಗಿದ್ದು, ಆದ್ದರಿಂದ ನಿಮ್ಮ ಪ್ರಯಾಣವನ್ನು ಪ್ರಾರಂಭದಿಂದ ಆರಂಭಿಸಿ ಕೊನೆಯಲ್ಲಿ ಮುಗಿಸುವುದನ್ನು ನಾವು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ.

ನೀವು AI ಏಜೆಂಟ್ ವಿಷಯಗಳ ಬಗ್ಗೆ ಇನ್ನಷ್ಟು ತಪಾಸಣೆ ಮಾಡಲು ಬಯಸಿದರೆ, ನೀವು [AI ಏಜೆಂಟ್‌ಗಳಿಗೆ ಪ್ರಾರಂಭಿಕರ ಕೋರ್ಸ್](https://aka.ms/ai-agents-beginners) ಅನ್ನು ಪರಿಶೀಲಿಸಬಹುದು.

### ಇತರ ಕಲಿಕಾರಂಗದವರನ್ನು ಭೇಟಿಯಾಗಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ ಪಡೆಯಿರಿ

ನೀವು ಅಡಿದಡೆ ಇರುವಾಗ ಅಥವಾ AI ಏಜೆಂಟ್‌ಗಳ ನಿರ್ಮಾಣ ಕುರಿತು ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ ಸಮರ್ಪಿತ Discord ಚಾನೆಲ್ನಲ್ಲಿ ಸೇರಿ [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) ನಲ್ಲಿ.

### ನಿಮಗೆ ಬೇಕಾದುದು

ಪ್ರತಿಯೊಂದು ಪಾಠದಕ್ಕೆ ಸ್ವಂತ ಕೋಡ್ ಮಾದರಿ ಇದೆ, ಅದನ್ನು ನೀವು ಸ್ಥಳೀಯವಾಗಿ ಚಲಾಯಿಸಬಹುದು. ನೀವು [ಈ ರೆಪೊವನ್ನು ಫೋರ್ಕ್](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) ಮಾಡಿ ನಿಮ್ಮ ನಕಲನ್ನು ರಚಿಸಬಹುದು.

ಈ ಕೋರ್ಸ್ ಈಗಲೂ ಕೆಳಗಿನವುಗಳನ್ನು ಬಳಸುತ್ತದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI ಸೇವೆ](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

ದಯವಿಟ್ಟು ಪ್ರಾರಂಭಿಸುವ ಮೊದಲು ಈ ಸೇವೆಗಳಿಗೆ ನಿಮ್ಮ ಪ್ರವೇಶವಿರುವುದನ್ನು ಖಾತ್ರಿಪಡಿಸಿ.

ಮಾದರಿ ನೆಡೆಯುವಿಕೆ ಮತ್ತು ಸೇವೆಗಳ ಮೇಲೆ ಇನ್ನಷ್ಟು ಆಯ್ಕೆಗಳು ತಲುಪಲಿವೆ.

ಈ ಯೋಜನೆ ಕೊಡುಗೆಗಳನ್ನು ಮತ್ತು ಸಲಹೆಗಳನ್ನು ಸ್ವಾಗತಿಸುತ್ತದೆ. ಹೆಚ್ಚಿನ ಕೊಡುಗೆಗಳಿಗೆ ನೀವು ಹಕ್ಕು ಹೊಂದಿರುವಿರಿ ಮತ್ತು ನಿಜವಾಗಿಯೂ ಕೊಡುಗೆಗಳನ್ನು ಬಳಸಲು ನಮಗೆ ಹಕ್ಕು ನೀಡುತ್ತೀರಿ ಎಂಬುದನ್ನು ಘೋಷಿಸುವ ಕೊಡುಗಿದಾರರ ಅನುಮತಿ ಒಪ್ಪಂದ (CLA) ಅನ್ನು ಒಪ್ಪಿಗೆ ನೀಡಬೇಕಾಗುತ್ತದೆ. ವಿವರಗಳಿಗೆ, <https://cla.opensource.microsoft.com> ಅನ್ನು ಭೇಟಿ ಮಾಡಿ.

ನೀವು ಪುಲ್ ರಿಕ್ವೆಸ್ಟ್ ಸಲ್ಲಿಸುವಾಗ, CLA ಬಾಟ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನೀವು CLA ಒದಗಿಸಲು ಅಗತ್ಯವಿದೆಯೇ ಅಥವಾ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ನಿರ್ಧರಿಸಿ, PR ಯನ್ನು ಸರಿಯಾಗಿ ಅಲಂಕರಿಸುತ್ತದೆ (ಉದಾ: ಸ್ಥಿತಿ ಪರಿಶೀಲನೆ, ಕಾಮೆಂಟ್). ಬಾಟ್ ನೀಡುವ ಸೂಚನೆಗಳನ್ನು ಅನುಸರಿಸಿ. ನಮ್ಮ CLA ಬಳಸಿ ಎಲ್ಲಾ ರೆಪોઝ್‌ಗಳಾದರಡಿಯೇ ನಿಮಗೆ ಇದು ಒಂದೇ ಬಾರಿ ಮಾತ್ರ ಮಾಡಬೇಕಾಗುತ್ತದೆ.
