## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್ಗಳ ನಿರ್ಮಾಣ ಮತ್ತು ಪ್ರಾರಂಭ ಮಾಡುವ ಮೂಲತತ್ವಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳನ್ನು ನೀಡುತ್ತದೆ.

ಪ್ರತಿ ಪಾಠವು ಹಿಂದಿನ ಪಾಠದ ಮೇಲೆ ನಿರ್ಮಾಣವಾಗಿದ್ದು ಹಾಗಾಗಿ ನಾವು ಆರಂಭದಿಂದ ಪ್ರಾರಂಭಿಸಿ ಕೊನೆವರೆಗೆ ವಿಚಾರಸರಣಿ ಮೂಲಕ ಹೋಗುವಂತೇ ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ.

ನೀವು AI ಏಜೆಂಟ್ ವಿಷಯಗಳನ್ನು ಇನ್ನಷ್ಟು ಅನ್ವೇಷಿಸಲು ಬಯಸಿದರೆ, ನೀವು [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) ಅನ್ನು ಪರಿಶೀಲಿಸಬಹುದು.

### ಇತರ ಕಲಿಕೆದಾರರನ್ನು ಭೇಟಿ ಮಾಡಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರಗಳನ್ನು ಪಡೆಯಿರಿ

ನೀವು ಅಡ್ಡಿಯಾಗಿದ್ದರೆ ಅಥವಾ AI ಏಜೆಂಟ್ಗಳ ನಿರ್ಮಾಣದ ಕುರಿತು ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ ಬದ್ಧ Discord ಚಾನಲ್ ಅನ್ನು [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) ನಲ್ಲಿ ಸೇರಿಕೊಳ್ಳಿ.

### ನಿಮಗೆ ಅಗತ್ಯವಿರುವುದು

ಪ್ರತಿ ಪಾಠವು ನಿಮ್ಮ ಸ್ಥಳೀಯದಲ್ಲಿ ಓಡಿಸಲು ಸಾಧ್ಯವಿರುವ ಸ್ವಂತ ಕೋಡ್ ಮಾದರಿಯಿದೆ. ನೀವು [ಈ ರೆಪೊ ಅನ್ನು ಫೋರ್ಕ್ ಮಾಡಬಹುದು](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) ನಿಮ್ಮ ಸ್ವಂತ ಪ್ರತಿಯನ್ನು ಸೃಷ್ಟಿಸಲು.

ಈ ಕೋರ್ಸ್ હાલ ಕೆಳಗಿನವುಗಳನ್ನು ಬಳಸುತ್ತದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

ಶುರುವಾಯಿಸುವ ಮೊದಲು ನೀವು ಈ ಸೇವೆಗಳಿಗೆ ಪ್ರವೇಶ ಹೊಂದಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.

ಮಾದರಿ ಆತಿಥೇಯ ಮತ್ತು ಸೇವೆಗಳ ಬಗ್ಗೆ ಇನ್ನಷ್ಟು ಆಯ್ಕೆಗಳು ಸೀಘ್ರದಲ್ಲೇ ಬರುತ್ತಿವೆ.
