# ಪಾಠ 2 ಏಜೆಂಟ್ ಅಭಿವೃದ್ಧಿ

"ಸೊನ್ನೆಯಿಂದ ಉತ್ಪಾದನೆಗೆ AI ಏಜೆಂಟ್ ನಿರ್ಮಾಣ ಕೋರ್ಸ್"ರ ಎರಡನೇ ಪಾಠಕ್ಕೆ ಸ್ವಾಗತ!

ಈ ಪಾಠದಲ್ಲಿ ನಾವು ಆವರಿಸುವುದು:

- ನಮ್ಮ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರಚಿಸಲು ಉಪಕರಣಗಳು

- ನಮ್ಮ ಅಭಿವೃದ್ಧಿ ಸಂಪನ್ಮೂಲಗಳ ಸೆಟಪ್ ಸೂಚನೆಗಳು

- AI ಏಜೆಂಟ್ ಅಭಿವೃದ್ಧಿ ಉತ್ತಮ ಪದ್ಧತಿಗಳು

- ನಮ್ಮ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರಚಿಸಲು ಕೋಡ್ ವಾಕ್‌ತ್ರೂ

ನಮ್ಮ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರಚಿಸಲು ನಾವು ಬಳಸುವ ಉಪಕರಣಗಳನ್ನು ನೋಡುವುದರಿಂದ ಶುರು ಮಾಡೋಣ.

## ಉಪಕರಣಗಳು ಮತ್ತು ಸೆಟಪ್ ಸೂಚನೆಗಳು

### ಮೈಕ್ರೋಸಾಫ್ಟ್ ಫೌಂಡ್ರಿ

ಲಾಗ್ರ ಲ್ಯಾಂಗ್ವೇಜ್ ಮಾದರಿಗಳಿಗೆ (LLMs) ಪ್ರವೇಶಕ್ಕೆ ನಾವು [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ಅನ್ನು ಬಳಸುತ್ತಿದ್ದೇವೆ. ಫೌಂಡ್ರಿಯನ್ನು ಬಳಸುವುದಕ್ಕೆ ಖರ್ಚುಗಳಿರುತ್ತದೆ, ಆದ್ದರಿಂದ ನೀವು ಈಗಾಗಲೇ ಪ್ರವೇಶ ಹೊಂದಿದ್ದಿರದಿದ್ದರೆ ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಲು ಖಾತೆ ಸೆಟಪ್‌ಗಾಗಿ ಸೂಚನೆಗಳನ್ನು ಅನುಸರಿಸಿ.

### OpenAI ಮಾದರಿಗಳು

ಈ ಕೋರ್ಸ್‌ನ ಏಜೆಂಟ್ ಕೋಡ್ ಮಾದರಿ ಉದಾಹರಣೆಗಳು OpenAI ಮಾದರಿಗಳನ್ನು [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ಮೂಲಕ ಬಳಸಲು ಸೆಟಪ್ ಮಾಡಲಾಗಿದೆ.

ಫೌಂಡ್ರಿ ಬಳಸಿ ಮಾದರಿಯನ್ನು ನಿಯೋಜಿಸುವುದನ್ನು ತಿಳಿಯಲು ಈ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಬಳಸಿರಿ: [Foundry ಪೋರ್ಟಲ್‌ನಲ್ಲಿ Microsoft Foundry ಮಾದರಿಗಳನ್ನು ನಿಯೋಜಿಸುವುದು](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

ಈ ಕೋರ್ಸ್‌ಗಾಗಿ ಒಂದು GPT-4.1 ಅಥವಾ ನಂತರದ ಮಾದರಿಯನ್ನು ಆಯ್ಕೆಮಾಡಿ.

### ಮೈಕ್ರೋಸಾಫ್ಟ್ ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್

ಹಿಂದೆ ಹೇಳಿದಂತೆ, ನಾವು AI ಏಜೆಂಟ್‌ಗಳನ್ನು ರಚಿಸಲು ಮತ್ತು ಪ್ರಸರಣ ಮಾಡಲು [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ಅನ್ನು ಬಳಸುತ್ತೇವೆ.

ಮೈಕ್ರೋಸಾಫ್ಟ್ ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್ ಮತ್ತು ಅಗತ್ಯವಿರುವ ಇತರೆ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸಲು, ಈ ಪ್ರಾಜೆಕ್ಟಿನ ರೂಟ್ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಕೆಳಗಿನ ಕಾಮಂಡ್ ಅನ್ನು ನಿರ್ವಹಿಸಿ:

```bash
pip install -r requirements.txt
```

### .env ಚರಗಳನ್ನು ಸೆಟಪ್ ಮಾಡಿ

ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿನ ಕೋಡ್ ಮಾದರಿ ಉದಾಹರಣೆಗಳನ್ನು ನಿರ್ವಹಿಸಲು, ನೀವು ಈ ಪ್ರಾಜೆಕ್ಟಿನ ರೂಟ್ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ `.env` ಫೈಲ್ ರಚಿಸಬೇಕು.

ಸುಲಭಮನ ಮಾಡಲು, ನೀವು ನೀಡಲಾದ `.env.example` ಫೈಲ್ ಅನ್ನು ನಕಲಿಸಿ ಬಳಸಬಹುದು:

```bash
cp .env.example .env
```
