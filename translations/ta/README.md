## 🌱 துவங்குதல்

இந்த பாடநெறியில் AI ஏஜெண்டுகளை உருவாக்குதல் மற்றும் பிரயோகிக்கப்படுத்துதல் சார்ந்த அடிப்படைகளை உள்ளடக்கிய பாடங்கள் உள்ளன.

ஒவ்வொரு பாடமும் முந்தையதை அடிப்படையாகக் கொண்டிருக்கிறது, ஆகவே ஆரம்பத்தில் இருந்து தொடங்கி முடிவில் வரையான முறையில் செயல்படுமாறு நாங்கள் பரிந்துரைக்கிறோம்.

AI ஏஜென்ட் தொடர்புடைய மேலதிக விஷயங்களை ஆராய விரும்பினால், நீங்கள் [AI ஏஜென்ட்கள் தொடக்கர்களுக்கான பாடநெறி](https://aka.ms/ai-agents-beginners) ஐப் பார்க்கலாம்.

### மற்ற பயிலருடன் சந்திக்கவும், உங்கள் கேள்விகளுக்கு பதில்கள் பெறவும்

AI ஏஜெண்ட்கள் உருவாக்குவதில் முடங்கினால் அல்லது கேள்விகள் இருந்தால், எங்கள் நியமிக்கப்பட்ட Discord சேனலான [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) இல் சேர்ந்து பேசுங்கள்.

### தேவையானவை

ஒவ்வொரு பாடத்திற்கும் தனித்து இயற்றி பார்க்கக்கூடிய குறியீட்டு எடுத்துக்காட்டுகள் உள்ளன. உங்கள் சொந்த நகலையை உருவாக்க [இந்த repo-ஐ fork செய்யலாம்](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork).

இந்தப் பாடநெறியில் தற்போது பின்வரும் சிறப்புப் பயன்பாடுகள்/சேவைகள் பயன்படுத்தப்படுகின்றன:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

துவங்குவதற்கு முன்னர் இத்தகைய சேவைகளுக்கான அணுகல் உங்களிடம் உள்ளதா என்பதை உறுதி செய்யுங்கள்.

மாதிரி ஹோஸ்டிங் மற்றும் சேவைகள் தொடர்பான கூடுதல் விருப்பங்கள் விரைவில் வெளியாகும்.

## பங்களிப்பு

இந்த திட்டம் பங்களிப்புக்களையும் பரிந்துரைகளையும் வரவேற்கிறது.  பெரும்பாலான பங்களிப்புகளில், நீங்கள் Contributor License Agreement (CLA)-க்கு ஒப்புக்கொள்ள வேண்டும்; அது நீங்கள் உங்கள் பங்களிப்பை பயன்படுத்துவதற்கான உரிமையை கொண்டிருப்பதாகவும், உண்மையிலேயும் நமக்கு அந்த உரிமைகளை வழங்குகிறீர்கள் என்பதையும் தெரிவிக்கிறது. விவரங்களுக்கு, <https://cla.opensource.microsoft.com> ஐப் பார்க்கவும்.

நீங்கள் ஒரு pull request சமர்ப்பிக்கும் போது, ஒரு CLA bot தானாகவே நீங்கள் CLA வழங்க வேண்டியதா என்பதை தீர்மானித்து
ஒரு CLA-வை வழங்கவேண்டுமெனில் PR-ஐ உரியவாறு அலங்கரித்து (உதாரணம்: நிலை சரிபார்ப்பு, கருத்து) காட்டுகிறது. bot வழங்கும்
வழிமுறைகளை எளிதில் பின்பற்றவும். எங்கள் CLA-வை பயன்படுத்தும் அனைத்து repos-களிலும் இதை ஒருமுறை மட்டும் செய்யவேண்டும்.

இந்த திட்டம் [Microsoft திறந்த மூல நடத்தை கொள்கை](https://opensource.microsoft.com/codeofconduct/)ஐ ஏற்றுக்கொண்டுள்ளது.
மேலும் தகவலுக்கு [நடத்தை தொடர்பான அடிக்கடி கேட்கப்படும் கேள்விகள்](https://opensource.microsoft.com/codeofconduct/faq/)ஐப் பார்க்கவும் அல்லது கூடுதல் கேள்விகள் அல்லது கருத்துகளுக்காக [opencode@microsoft.com](mailto:opencode@microsoft.com)-ஐ தொடர்பு கொள்ளவும்.

## வர்த்தகச் சின்னங்கள்

இந்தத் திட்டத்தில் திட்டங்கள், தயாரிப்புகள் அல்லது சேவைகளுக்கான வர்த்தகச் சின்னங்கள் அல்லது லோகோக்கள் இடம்பெறலாம். Microsoft
வர்த்தகச் சின்னங்கள் அல்லது லோகோக்களின் அதிகாரபூர்வ பயன்பாடு [Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general)-க்கு உட்பட்டது மற்றும் அதனை கடைப்பிடிக்கவேண்டும்.
இந்தத் திட்டத்தின் மாற்றப்பட்ட பதிப்புகளில் Microsoft வர்த்தகச் சின்னங்கள் அல்லது லோகோக்களின் பயன்பாடு குழப்பத்தை உண்டாக்கவே கூடாது அல்லது Microsoft ஆதரவைக் குறிக்கும் வகையில் தோன்றக்கூடாது.
மூன்றாம் தரப்பு வர்த்தகச் சின்னங்கள் அல்லது லோகோக்களின் எந்தவொரு பயன்பாடும் அந்த மூன்றாம் தரப்பினரின் கொள்கைகளுக்கு உட்பட்டது.
