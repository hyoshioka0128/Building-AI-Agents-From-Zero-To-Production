## 🌱 பணிக்கு துவக்கம்

இந்த பாடநெறியில் ஏஐ முகவர்கள் கட்டமைப்பு மற்றும் நடத்தை அடிப்படைகள் உள்ள பாடங்கள் உள்ளன.

ஒவ்வொரு பாடமும் முந்தையதை தொடர்ந்தே உருவாக்கப்படுவதால், ஆரம்பத்திலிருந்து துவங்கி கடைசிவரை முறையாக வேலை செய்ய பரிந்துரைக்கப்படுகிறது.

நீங்கள் ஏஐ முகவர் தொடர்பான மேலும் ஆராய விரும்பினால், [AI Agents For Beginners Course](https://aka.ms/ai-agents-beginners) ஐ பார்க்கலாம்.

### மற்ற பயனர்களைப் சந்தித்து, உங்கள் கேள்விகளுக்கு பதில்கள் பெறுங்கள்

நீங்கள் பாதிக்கப்பட்டால் அல்லது ஏஐ முகவர்கள் உருவாக்குதலில் ஏதேனும் கேள்விகள் இருந்தால், [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6) இல் உள்ள எங்கள் தனித்துவமான Discord சேனலில் சேரவும்.

### நீங்கள் தேவையென்ன?

ஒவ்வொரு பாடத்திற்கும் தனி குறியீடு மாதிரி உள்ளது, அதை நீங்கள் உள்ளூரில் இயக்கலாம். [இந்த தொகுப்பகத்தை பணி கிளோன் செய்ய](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork) முடியும்.

இந்த பாடநெறி தற்போது கீழ்காணும் சேவைகளை பயன்படுத்துகிறது:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

துவங்குவதற்கு முன்பு இந்த சேவைகளுக்கான அணுகலை உறுதி செய்யவும்.

மாதிரி ஹோஸ்டிங் மற்றும் சேவைகள் தொடர்பான கூடுதல் விருப்பங்கள் விரைவில் வரும்.
