# பாடம் 2 முகவர் மேம்பாடு

"Building AI Agent from Zero to Production Course" என்ற இரண்டாவது பாடத்திற்கு வருக!

இந்த பாடத்தில் நாம் கையாளப்போகும் புள்ளிகள்:

- நமது AI முகவர்களை உருவாக்க பயன்படுத்தும் கருவிகள்
  
- மேம்பாட்டு வளங்களுக்கான அமைப்பு அறிவுரைகள்

- AI முகவர் மேம்பாட்டு சிறந்த நடைமுறைகள்
  
- நமது AI முகவர்களை உருவாக்கக் கூடிய குறியீடு நடைபயணம்
  
நாம் ஏற்கனவே பயன்படுத்தப்போகும் கருவிகளை பார்க்க ஆரம்பிக்கலாம்.

## கருவிகள் மற்றும் அமைப்பு அறிவுரைகள்

### Microsoft Foundry

பெரிய மொழி மாதிரிகளுக்கு (LLMs) அணுகல் பெற, நாம் [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) பயன்படுத்தும். Foundry பயன்படுத்துவதற்கு செலவுகள் உள்ளன, ஆகவே கணக்கு அமைக்கும் அறிவுரைகளை பின்பற்ற வேண்டும், நீங்கள் ஏற்கனவே அணுகலை இல்லையெனில்.

### OpenAI மாதிரிகள்

இந்த பாடத்திலுள்ள முகவர் குறியீடு உதாரணங்கள் [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) வழியாக OpenAI மாதிரிகளைப் பயன்படுத்த அமைக்கப்பட்டுள்ளன.

Foundry வழியாக மாதிரியை Nasari செய்ய கற்றுக் கொள்ள இந்த வழிகாட்டியைப் பயன்படுத்தவும்: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

இந்த பாடத்திற்காக GPT-4.1 அல்லது அதற்குப் பின்வரும் மாதிரி ஒன்றைத் தேர்ந்தெடுக்கவும்.

### Microsoft Agent Framework

முன்பே குறிப்பிடப்பட்டபடி, நமது AI முகவர்களை உருவாக்கவும் ஒருங்கிணைக்கவும் [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) பயன்படுத்தப்போகிறோம்.

Microsoft Agent Framework மற்றும் பிற தேவையான தொகுப்புகளை நிறுவ, இந்த திட்டத்தின் ரூட் அடைவிலிருக்கும் போது கீழ்க்காணும் கட்டளை ஓட்டவும்:

```bash
pip install -r requirements.txt
```

### .env மாறிலிகளை அமைத்தல்

இந்த பாடத்தில் குறியீடு உதாரணங்களை ஓட்ட, இந்த திட்டத்தின் ரூட் அடைவில் `.env` கோப்பை உருவாக்க வேண்டும்.

இதை எளிதாக்க, வழங்கப்பட்ட `.env.example` கோப்பின் நகலை எடுத்துக்கொள்ளலாம்:

```bash
cp .env.example .env
```
