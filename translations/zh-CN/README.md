## 🌱 入门指南

本课程涵盖构建和部署AI代理的基础课程。

每节课都是建立在前一节的基础上，因此建议从头开始，逐步完成所有课程。

如果你想了解更多AI代理相关话题，可以查看[AI代理初学者课程](https://aka.ms/ai-agents-beginners)。

### 认识其他学习者，获取问题解答

如果你遇到困难或对构建AI代理有任何疑问，欢迎加入我们的专属Discord频道，[Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6)。

### 你需要准备的东西

每节课都有对应的代码示例，你可以本地运行。你可以[叉这个仓库的分支](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)来创建你自己的副本。

本课程当前使用以下服务：

- [Microsoft代理框架(MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI服务](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

请确保在开始前你已能访问这些服务。

后续将推出更多模型托管及服务选项。
