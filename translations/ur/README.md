## 🌱 شروعات

اس کورس میں اسباق شامل ہیں جو AI ایجنٹس کی تعمیر اور ڈپلائے کرنے کے بنیادی اصول سکھاتے ہیں۔

ہر سبق پچھلے سبق پر مبنی ہوتا ہے، لہٰذا ہم سفارش کرتے ہیں کہ ابتدا سے شروع کریں اور آخر تک جائیں۔

اگر آپ AI ایجنٹس کے موضوعات کے بارے میں مزید جاننا چاہتے ہیں تو آپ [AI ایجنٹس برائے ابتدائی افراد کورس](https://aka.ms/ai-agents-beginners) دیکھ سکتے ہیں۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات کے جواب حاصل کریں

اگر آپ کہیں پھنس جائیں یا AI ایجنٹس بنانے کے بارے میں کوئی سوال ہو تو ہمارے مخصوص ڈسکارڈ چینل میں شامل ہوں: [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6).

### آپ کو کیا چاہیے

ہر سبق کے ساتھ اس کا اپنا کوڈ نمونہ ہے جسے آپ مقامی طور پر چلا سکتے ہیں۔ آپ اپنی کاپی بنانے کے لیے [اس ریپوزیٹری کو فورک کریں](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)۔

یہ کورس فی الحال مندرجہ ذیل استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)
- [Azure CLI](https://learn.microsoft.com/cli/azure/authenticate-azure-cli?view=azure-cli-latest)

براہِ کرم شروعات سے پہلے یقینی بنائیں کہ آپ کو ان سروسز تک رسائی حاصل ہے۔

ماڈل ہوسٹنگ اور سروسز کے حوالے سے مزید اختیارات جلد آ رہے ہیں۔
