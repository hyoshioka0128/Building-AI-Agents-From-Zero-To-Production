## 🌱 ਸ਼ੁਰੂਆਤ ਕਰਨਾ

ਇਸ ਕੋਰਸ ਵਿੱਚ ਏਆਈ ਏਜੰਟ ਬਣਾਉਣ ਅਤੇ ਤਾਇਨਾਤ ਕਰਨ ਦੇ ਮੁਢਲੇ ਸਿਧਾਂਤ ਨੂੰ ਸਮਝਾਉਂਦੇ ਪਾਠ ਹਨ।

ਹਰ ਪਾਠ ਪਿਛਲੇ ਪਾਠ 'ਤੇ ਆਧਾਰਿਤ ਹੈ, ਇਸ ਲਈ ਅਸੀਂ ਸਿਫਾਰਿਸ਼ ਕਰਦੇ ਹਾਂ ਕਿ ਸ਼ੁਰੂ ਤੋਂ ਸ਼ੁਰੂ ਕਰਕੇ ਅਖੀਰ ਤੱਕ ਅੱਗੇ ਵਧੋ।

ਜੇ ਤੁਸੀਂ ਏਆਈ ਏਜੰਟ ਵਿਸ਼ਿਆਂ ਬਾਰੇ ਹੋਰ ਜਾਣਨ ਦੇ ਇਚਛੁਕ ਹੋ, ਤਾਂ ਤੁਸੀਂ [AI Agents For Beginenrs Course](https://aka.ms/ai-agents-beginners) ਚੈੱਕ ਕਰ ਸਕਦੇ ਹੋ।

### ਹੋਰ ਸਿੱਖਣ ਵਾਲਿਆਂ ਨਾਲ ਮਿਲੋ, ਆਪਣੇ ਸਵਾਲਾਂ ਦੇ ਉੱਤਰ ਲਵੋ

ਜੇ ਤੁਸੀਂ ਫਸ ਜਾਂਦੇ ਹੋ ਜਾਂ ਏਆਈ ਏਜੰਟ ਬਣਾਉਣ ਬਾਰੇ ਕੋਈ ਪ੍ਰਸ਼ਨ ਹੈ, ਤਾਂ ਸਾਡੀ ਸਮਰਪਿਤ ਡਿਸਕੋਰਡ ਚੈਨਲ ਵਿੱਚ ਸ਼ਾਮਲ ਹੋਵੋ [Microsoft Foundry Discord](https://discord.gg/Kuaw3ktsu6)।

### ਤੁਹਾਨੂੰ ਕੀ ਚਾਹੀਦਾ ਹੈ

ਹਰ ਪਾਠ ਦਾ ਆਪਣਾ ਕੋਡ ਨਮੂਨਾ ਹੈ ਜੋ ਤੁਸੀਂ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਚਲਾ ਸਕਦੇ ਹੋ। ਤੁਸੀਂ ਆਪਣੀ ਕਾਪੀ ਬਣਾਉਣ ਲਈ [ਕਲੋਨ ਕਰਕੇ ਫੋਰਕ ਕਰ ਸਕਦੇ ਹੋ](https://github.com/microsoft/Building-AI-Agents-From-Zero-To-Production/fork)।

ਇਸ ਕੋਰਸ ਵਿੱਚ ਹਾਲ ਹੀ ਵਿੱਚ ਹੇਠ ਲਿਖੇ ਵਰਤੋਂ ਵਿੱਚ ਹਨ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry)
- [Azure OpenAI Service](https://azure.microsoft.com/products/ai-foundry/models/openai)

ਕਿਰਪਾ ਕਰਕੇ ਸ਼ੁਰੂ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਇਹ ਸੇਵਾਵਾਂ ਉਪਲੱਬਧ ਹਨ ਜਾਂ ਨਹੀਂ, ਇਸ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ।

ਮਾਡਲ ਹੋਸਟਿੰਗ ਅਤੇ ਸੇਵਾਵਾਂ ਸੰਬੰਧੀ ਹੋਰ ਵਿਕਲਪ ਜਲਦੀ ਆ ਰਹੇ ਹਨ।
