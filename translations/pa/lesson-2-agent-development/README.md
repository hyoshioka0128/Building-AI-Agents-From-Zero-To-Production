# ਪਾਠ 2 ਏਜੰਟ ਵਿਕਾਸ

"ਸ਼ੂਨ ਤੋਂ ਪ੍ਰੋਡਕਸ਼ਨ ਤੱਕ ਤਿਆਰ ਕੀਤਾ ਜਾ ਰਿਹਾ ਏਆਈ ਏਜੰਟ" ਕੋਰਸ ਦੇ ਦੂਜੇ ਪਾਠ ਵਿੱਚ ਤੁਹਾਡਾ ਸੁਆਗਤ ਹੈ!

ਇਸ ਪਾਠ ਵਿੱਚ ਅਸੀਂ ਕਵਰ ਕਰਾਂਗੇ:

- ਆਪਣੇ ਏਆਈ ਏਜੰਟ ਬਣਾਉਣ ਲਈ ਸੰਦ
  
- ਆਪਣੇ ਵਿਕਾਸ ਸਰੋਤਾਂ ਲਈ ਸੈਟਅਪ ਹدایਤਾਂ

- ਏਆਈ ਏਜੰਟ ਵਿਕਾਸ ਦੀਆਂ ਸਭ ਤੋਂ ਵਧੀਆ ਅਭਿਆਸ
  
- ਆਪਣੇ ਏਆਈ ਏਜੰਟ ਬਣਾਉਣ ਲਈ ਕੋਡ ਵਾਲਕਥਰੂ
  
ਆਓ ਸ਼ੁਰੂ ਕਰੀਏ ਇਹ ਦੇਖ ਕੇ ਕਿ ਅਸੀਂ ਆਪਣੇ ਏਜੰਟ ਬਣਾਉਣ ਲਈ ਕਿਹੜੇ ਸੰਦ ਵਰਤਾਂਗੇ।

## ਸੰਦ ਅਤੇ ਸੈਟਅਪ ਹدایਤਾਂ

### Microsoft Foundry

ਵੱਡੇ ਭਾਸ਼ਾ ਮਾਡਲਾਂ (LLMs) ਤੱਕ ਪਹੁੰਚ ਲਈ ਅਸੀਂ [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ਦੀ ਵਰਤੋਂ ਕਰਾਂਗੇ। Foundry ਦੀ ਵਰਤੋਂ ਨਾਲ ਸੰਬੰਧਿਤ ਖਰਚ ਹਨ, ਇਸ ਲਈ ਜੇ ਤੁਹਾਡੇ ਕੋਲ ਪਹਿਲਾਂ ਤੋਂ ਪਹੁੰਚ ਨਹੀਂ ਹੈ ਤਾਂ ਕਿਰਪਾ ਕਰਕੇ ਖਾਤਾ ਸੈਟਅਪ ਲਈ ਦਿੱਤੀਆਂ ਹدایਤਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ।

### OpenAI ਮਾਡਲਸ

ਇਸ ਕੋਰਸ ਵਿੱਚ ਏਜੰਟ ਕੋਡ ਨਮੂਨੇ [Microsoft Foundry](https://azure.microsoft.com/products/ai-foundry) ਰਾਹੀਂ OpenAI ਮਾਡਲਾਂ ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ ਸੈਟਅਪ ਕੀਤੇ ਗਏ ਹਨ।

Foundry ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕਿਸੇ ਮਾਡਲ ਨੂੰ ਡਿਪਲੋਇ ਕਰਨ ਲਈ ਇਸ ਮਾਰਗਦਰਸ਼ਨ ਨੂੰ ਵਰਤੋਂ: [Deploy Microsoft Foundry Models in the Foundry portal](https://learn.microsoft.com/azure/ai-foundry/foundry-models/how-to/deploy-foundry-models?view=foundry-classic)

ਇਸ ਕੋਰਸ ਲਈ GPT-4.1 ਜਾਂ ਉਸ ਤੋਂ ਵੱਧ ਦਾ ਕੋਈ ਇੱਕ ਮਾਡਲ ਚੁਣੋ।

### Microsoft Agent Framework

ਜਿਵੇਂ ਪਹਿਲਾਂ ਦੱਸਿਆ ਗਿਆ ਹੈ, ਅਸੀਂ ਆਪਣੇ ਏਆਈ ਏਜੰਟ ਬਣਾਉਣ ਅਤੇ ਸੰਚਾਲਿਤ ਕਰਨ ਲਈ [Microsoft Agent Framework](https://github.com/microsoft/agent-framework) ਦੀ ਵਰਤੋਂ ਕਰਾਂਗੇ।

Microsoft Agent Framework ਅਤੇ ਹੋਰ ਜਰੂਰੀ ਪੈਕੇਜਾਂ ਨੂੰ ਇੰਸਟਾਲ ਕਰਨ ਲਈ, ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਰੂਟ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਹੋ ਕੇ ਹੇਠਾਂ ਦਿੱਤਾ ਕਮਾਂਡ ਚਲਾਓ:

```bash
pip install -r requirements.txt
```

### .env ਵੈਰੀਏਬਲ ਸੈਟਅਪ

ਇਸ ਕੋਰਸ ਵਿੱਚ ਕੋਡ ਦੇ ਨਮੂਨੇ ਚਲਾਉਣ ਲਈ, ਤੁਹਾਨੂੰ ਪ੍ਰੋਜੈਕਟ ਦੀ ਰੂਟ ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਇੱਕ `.env` ਫਾਈਲ ਬਣਾਉਣੀ ਪਵੇਗੀ।

ਇਹ ਆਸਾਨ ਬਣਾਉਣ ਲਈ, ਤੁਸੀਂ ਦਿੱਤੀ ਗਈ `.env.example` ਫਾਈਲ ਨੂੰ ਕਾਪੀ ਕਰ ਸਕਦੇ ਹੋ:

```bash
cp .env.example .env
```
